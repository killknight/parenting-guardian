# 支付宝云环境配置指南

## 一、登录支付宝云控制台

1. 访问 [支付宝云控制台](https://home.cloud.alipay.com/)
2. 使用支付宝账号登录
3. 选择服务空间：`parenting-guardian` (SpaceID: `env-00jy6g81zqkm`)

## 二、创建数据库集合

### 步骤1：进入数据库管理
1. 在左侧菜单中找到 **云数据库**
2. 点击 **创建数据库**（如果提示）

### 步骤2：逐个创建集合

按以下顺序创建9个集合：

#### 2. devices（设备表）
```
集合名称：devices
```

#### 3. bindings（绑定关系表）
```
集合名称：bindings
```

#### 4. usage_records（使用记录表）
```
集合名称：usage_records
```

#### 5. location_records（位置记录表）
```
集合名称：location_records
```

#### 6. commands（指令表）
```
集合名称：commands
```

#### 7. alerts（告警表）
```
集合名称：alerts
```

#### 8. geofences（围栏表）
```
集合名称：geofences
```

#### 9. geofence_events（围栏事件表）

```
集合名称：geofence_events
```

### 步骤3：设置集合权限

每个集合创建后，点击集合名称，进入 **权限设置**，建议设置为：
```json
{
  "read": "auth.uid != null",
  "create": "auth.uid != null",
  "update": "auth.uid != null",
  "delete": "auth.uid != null"
}
```

## 三、上传云函数

### 方式一：通过支付宝云控制台上传

1. 在左侧菜单中找到 **云函数**
2. 点击 **创建云函数**
3. 选择上传方式：**上传文件夹** 或 **上传代码包**
4. 选择 `/workspace/parenting-guardian-cloud` 文件夹
5. 填写云函数名称：`cloudfunctions`
6. 填写入口函数：`index.main`
7. 点击 **部署**

### 方式二：通过HBuilderX上传

1. 用HBuilderX打开项目
2. 在项目管理器中右键点击 `parenting-guardian-cloud`
3. 选择 **上传云函数/云对象**
4. 等待上传完成

### 云函数结构说明

```
parenting-guardian-cloud/
├── index.js              # 主入口，统一路由
├── database.js           # 数据库操作入口
├── auth/                 # 认证模块
│   ├── login.js
│   ├── register.js
│   └── sendCode.js
├── user/                 # 用户模块
├── binding/              # 绑定模块
├── device/               # 设备模块
├── location/             # 位置模块
├── usage/                # 使用统计模块
├── command/              # 指令模块
├── alert/                # 告警模块
└── geofence/             # 围栏模块
```

## 四、在HBuilderX中关联云空间

### 步骤1：获取云空间信息
- SpaceID: `env-00jy6g81zqkm`
- SpaceName: `parenting-guardian`
- Provider: `aliyun`

### 步骤2：在HBuilderX中配置

1. 打开HBuilderX
2. 打开 `parenting-guardian-app` 项目
3. 点击顶部菜单：**运行** → **运行到手机或模拟器** → **运行设置**
4. 在 **云函数/云对象空间** 中选择对应的云空间
5. 或者在项目根目录的 `manifest.json` 中已配置：

```json
"uniCloud": {
  "spaceId": "env-00jy6g81zqkm",
  "spaceProvider": "aliyun",
  "spaceName": "parenting-guardian"
}
```

### 步骤3：初始化云数据库

1. 在HBuilderX中右键点击项目
2. 选择 **云函数** → **初始化云数据库**
3. 系统会自动创建集合

## 六、验证配置

### 测试云函数

1. 在支付宝云控制台的云函数页面
2. 点击刚上传的云函数
3. 选择 **测试函数**
4. 输入测试参数：
```json
{
  "action": "auth.sendCode",
  "data": {
    "phone": "13800138000"
  }
}
```
5. 点击 **执行**，查看返回结果

### 测试数据库

1. 在云数据库页面
2. 选择 `users` 集合
3. 点击 **添加记录**
4. 手动插入一条测试数据验证连接

## 七、常见问题

### Q: 云函数上传失败怎么办？
A: 检查网络连接，确保云空间状态正常，或尝试分批上传子模块。

### Q: 数据库权限如何设置？
A: 生产环境建议按需设置权限，以上配置为开发环境设置。

### Q: 如何查看云函数日志？
A: 在云函数页面点击函数名称，选择 **日志** 标签查看。
